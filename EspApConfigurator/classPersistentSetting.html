virtual bool PersistentSetting::set ( String  newValue)
pure virtual

Set the value from a String.

If the specified value fails the isValid(newValue) test, the value of the setting will remain unchanged. This only sets the value in RAM - to save to EEPROM, save() must be called.

Returns
true if the specified value was set, else false
