bool PersistentSettingString::isValid ( String  value)

Validation.

Determine if the current value is valid - used after loading from EEPROM and when setting the value. If there is no validator function specified for this setting (i.e. validator is NULL), any value is consifered valid. If validator is not NULL, it will used to determine validite of the value being tested. valid/invalid value.

Returns
true if value is valid, else false
